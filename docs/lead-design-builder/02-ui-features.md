# CRM Lead Design Builder - UI & Layout Features

## Builder Layout
```text
 ---------------------------------------------------------
| Top Toolbar                                              |
| Save | Export | Undo | Redo | Preview | Templates       |
 ---------------------------------------------------------
| Layers | Canvas Area | Properties Panel                 |
|        |             |                                  |
|        |             | Font                             |
|        |             | Colors                           |

 ---------------------------------------------------------
```

## Canvas Features
### Multiple Canvas Sizes
Supported sizes:
- Portrait
- Landscape
- Square
- A4
- Letter
- Story
- 4:3
- 16:9
- Custom Size (e.g., 1080x1080, 1080x1350, 1920x1080, 1080x1920, 1600x1200)
Users can switch sizes anytime.

## Layer Management
Canva-style layers panel.
Features:
- Bring Forward / Send Backward
- Bring To Front / Send To Back
- Lock Layer / Unlock Layer
- Hide Layer
- Duplicate Layer
- Delete Layer

## Background Controls
Canvas Background:
- Solid Color
- Gradient
- Image Background
- Pattern Background
Options: Blur, Brightness, Opacity

## Preview Mode
Preview: Desktop, Tablet, Mobile

## Undo / Redo
Keyboard Shortcuts:
- Ctrl + Z (Undo)
- Ctrl + Y (Redo)
