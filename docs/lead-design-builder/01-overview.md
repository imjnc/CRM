# CRM Lead Design Builder - Overview & Phases

## Overview
Add a new action button called "Build" on every Lead row in the CRM.

### Leads Page
| Lead Name | Company | Email | Phone | Actions |
| --------- | ------- | ----- | ----- | ------- |
| John Doe  | ABC Ltd | john@abc.com | 9999999999 | Build |

When the user clicks Build: `/leads/{leadId}/build`
The user is redirected to a Canva-style design builder page where they can create proposals, flyers, brochures, social media creatives, quotations, presentations, landing pages, marketing materials, and lead-specific documents.

## Main Goals
1. Create custom designs
2. Auto-fill CRM lead information
3. Export designs
4. Save designs
5. Reuse templates
6. Share designs
7. Maintain design history
8. Improve sales and marketing workflow
